Цель и задачи

Цель: Спроектировать и разработать прототип интерактивного гида по японской мифологии.

Задачи:

  • Проанализировать японскую мифологию
  • Спроектировать структуру сайта
  • Создать дизайн-концепцию в Figma
  • Разработать адаптивный интерфейс
  • Опубликовать проект на GitHub Pages

Отсутствие доступных русскоязычных ресурсов, сочетающих энциклопедические знания о японской мифологии с современными интерактивными технологиями.

Существующие решения либо статичны, либо требуют установки приложений.
